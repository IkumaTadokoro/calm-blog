Vitest UIを試してみる

```bash
npm i -D @vitest/ui
```

これで

```json:package.json
    "test:ui": "vitest --ui",
```

こう。で動いた！たぶんテスト対象のファイルだけを抽出してくれるので、srcと同じディレクトリにテストを置く方針で、テストが増えまくった時に便利そう
