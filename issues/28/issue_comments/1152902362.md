https://vuejs.org/guide/essentials/template-refs.html

- `ref`をHTML要素の中で使用すると、インスタンスへの」直接の参照を取得することができる。

## Accessing the Refs

```vue
// inputという名前で参照を取得する
<input ref = "input" />

<script setup>
import { ref, onMounted } from 'vue'

const input = ref(null)

onMounted(() => {
  input.value.focus()
})
</script>
```
